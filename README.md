# Vue Hello

## 安裝

```
yarn global add @vue/cli @vue/cli-service-global
```

```
yarn global upgrade --latest @vue/cli
```

## 建立專案

```
vue create hello-world
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
